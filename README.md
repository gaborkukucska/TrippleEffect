## Quick Start
```bash
# Clone repository
git clone https://github.com/gaborkukucska/TrippleEffect.git
cd TrippleEffect
```

# Run setup script
```bash
bash setup.sh
```

# Start application
```bash
python main.py
```

3. Set API keys and model parameters
4. Configure sandbox environments
5. Save and initialize agents
